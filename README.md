# Secure Cloud Backup and Recovery Tool (Cassandra Assignment)

## Project Overview
This project demonstrates cloud backup logging using Apache Cassandra  
with Docker for database deployment.

## Features
- Snapshot logging  
- Backup status tracking  
- Querying backup metadata  
- Cassandra table design  
- Real-world use case for community centers  

## Included Files
- assignment.docx – Complete documentation
- queries.cql – All CQL queries used
- screenshots/ – Screenshots of executed queries
- commands.txt – Docker + Cassandra commands
